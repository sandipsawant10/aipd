# 🤖 AI Product Development Lab Assignments

This repository contains practical lab assignments for the **AI Product Development** course, focusing on building real-world AI systems using core concepts of Artificial Intelligence, Machine Learning, and Deep Learning.

---

## 📚 Overview

The labs cover a wide range of AI techniques including:

* 🔍 Search Algorithms
* 🧠 Logic & Knowledge-Based Systems
* 📊 Machine Learning Models
* 🤖 Neural Networks & Deep Learning

Each assignment is implemented using **Python** with a focus on practical understanding and problem-solving.



## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** NumPy, Pandas, Scikit-learn, TensorFlow / PyTorch
* **Environment:** Jupyter Notebook / Python Scripts
